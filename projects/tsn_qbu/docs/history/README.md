# Historical Qbu Records

이 디렉터리는 날짜별 시험 당시의 관찰과 가설을 보존한다. 현재 setup, pass/fail 기준,
재현 절차, project close 판단에는 사용하지 않는다.

현재 canonical 문서:

- `../CLEAN_BASE_CONTRACT.md`
- `../VALIDATION_STATUS.md`
- `../REPRODUCTION.md`
- `../PROVENANCE.md`
- `../CLOSURE_CHECKLIST.md`

이력 문서의 당시 결론이 canonical 문서와 충돌하면 canonical 문서가 우선한다.
