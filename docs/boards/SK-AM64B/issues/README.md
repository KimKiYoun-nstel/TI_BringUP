## 운용 원칙

- 긴 조사 과정, 가설, 원문 인용, 세부 비교는 `docs/research/` 에 둔다.
- 현재 진행 중인 액션 아이템은 `docs/tasks/` 에 둔다.
- 여기에는 **해결되었거나 상태가 정리된 보드별 이슈의 짧은 요약본**만 둔다.
- 각 이슈 문서는 가능하면 다음 항목을 유지한다.
  - 증상
  - 원인
  - 조치
  - 검증
  - 후속 메모
