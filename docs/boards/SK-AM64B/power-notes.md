# SK-AM64B 전원 노트

## 목적

이 문서는 첫 번째 SK-AM64B 부팅 세션에서 확인한 전원 관련 관찰 사항을 기록합니다.

## 전원 입력

SK-AM64B는 USB-C로 전원을 공급받습니다.

초기 bring-up 단계에서는 안정적인 USB-C PD 전원을 사용하는 것이 좋습니다.

## 관찰 내용

초기 전원:

```text
25W USB-C 전원 어댑터
```

관찰된 문제:

```text
보드 전원 상태가 bring-up 작업을 진행하기에 충분히 안정적이지 않았음
```

변경한 전원:

```text
65W 노트북용 USB-C 전원 어댑터
```

결과:

```text
SD 부팅과 Linux 로그인까지 진행할 수 있을 만큼 전원이 안정화됨
```

## 결정

초기 SK-AM64B bring-up 작업의 기본 전원으로 65W 노트북 USB-C 어댑터를 사용합니다.

## 실전 규칙

초기 보드 bring-up에서는 여유가 부족한 전원을 피합니다.

권장:

```text
정상 동작이 확인된 USB-C PD 어댑터
충분한 전류 여유
가능하면 직접 연결
안정적인 케이블
```

지양:

```text
출처가 불분명한 충전기
저가형 USB-A to USB-C 충전 경로
신뢰하기 어려운 USB-C 케이블
전력이 부족한 허브/전원 공급원
```

## 전원 의심 시 디버그 순서

보드 동작이 불안정해 보이면:

```text
1. USB-C 전원 어댑터 교체
2. USB-C 케이블 교체
3. 보드 전원 LED 확인
4. 불필요한 주변장치 제거
5. UART COM 포트 인식 재확인
6. SD 부팅 재확인
```

## Bring-up 해석

전원 문제는 소프트웨어 문제처럼 보일 수 있습니다.

전원이 불안정할 때 가능한 증상:

```text
UART 로그가 없음
부팅이 일부만 진행됨
무작위 재부팅
SD 카드 읽기 오류
USB 장치가 사라짐
네트워크/Wi-Fi 불안정
```

따라서 bootloader, kernel, rootfs를 깊게 디버깅하기 전에 전원부터 검증해야 합니다.
